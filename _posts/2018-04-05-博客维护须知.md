博客维护
===

### 上传一个博客的git命令

    添加文件

        git add filename.md
        git add -A .

    删除文件

        git rm filename.md

    添加注释

        git commit -m 'modify filename.md'

    提交文件

        git push

### 本地运行服务器的命令

        jekyll serve
